# 粒时 隐私政策

**最后更新日期**: 2026-06-08
**生效日期**: 2026-06-08

您的隐私对我们至关重要。本隐私政策说明 **"粒时"** 移动应用 (以下简称"本应用") 如何处理您的信息。请您仔细阅读本政策, 一旦您开始使用本应用即视为您已阅读、理解并接受本政策的全部内容。

本政策依据《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《App 违法违规收集使用个人信息行为认定方法》等相关法律法规制定。

---

## 一、我们收集的信息

### 1.1 核心原则

**本应用不收集任何个人信息**。

"个人信息"是指以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息, 不包括匿名化处理后的信息。

### 1.2 具体说明

本应用**不收集**以下任何信息:

| 类别 | 是否收集 | 说明 |
|---|---|---|
| 身份信息 (姓名、手机号、身份证号) | ❌ 不收集 | 不需要注册账号 |
| 位置信息 (精确 / 粗略) | ❌ 不收集 | 不申请位置权限 |
| 通讯录 / 短信 / 通话记录 | ❌ 不收集 | 不申请相关权限 |
| 设备识别码 (IMEI / MAC / Android ID / OAID) | ❌ 不收集 | 不申请相关权限 |
| 相机 / 麦克风 / 相册 | ❌ 不访问 | 不申请相关权限 |
| 传感器数据 | ❌ 不收集 | 不主动读取 |
| 行为日志 / 使用统计 | ❌ 不收集 | 未集成任何分析 SDK |
| 崩溃 / 错误信息 | ❌ 不上报 | 未集成任何崩溃 SDK |
| 网络请求 | ❌ 不发起 | 应用运行期间不联网 |

---

## 二、信息的存储

### 2.1 存储位置

您在粒时中创建的所有用户内容 (如"读了 30 页书"、用时 25 分钟等提交记录) **仅存储在您的设备本地**, 具体位置为 Android 系统的应用沙箱目录 (即 `/data/data/com.example.lishi/databases/lishi.db` 之类的私有目录)。

### 2.2 存储期限

| 情形 | 后果 |
|---|---|
| 正常使用 | 数据保留在设备本地, 长期保留 |
| 您在应用内删除单条记录 | 该条数据立即从本地删除 |
| 您在系统设置中清除应用数据 | **所有**本地数据被清除 |
| 您卸载本应用 | **所有**本地数据被永久删除, 且**无法恢复** |

### 2.3 我们的访问能力

**我们没有任何后端服务器, 因此我们无法访问、查看、修改、删除或恢复您的任何本地数据**。您的数据完全处于您自己的控制之下。

---

## 三、信息的使用

由于本应用**不收集任何信息**, 因此不存在"使用"环节。

您在本地创建的数据, 仅为实现本应用的"提交记录"功能而被本应用读取和展示。

---

## 四、信息的共享、转让、公开披露

由于本应用**不收集任何信息**, 因此不存在"共享、转让、公开披露"环节。

我们**承诺**:
- **不向任何第三方共享** 您的任何信息 (因为我们没有您的信息)
- **不向任何第三方转让** 您的任何信息
- **不公开披露** 您的任何信息
- **不进行任何形式的用户画像、行为分析、广告推送**

---

## 五、第三方 SDK 与服务

本应用**未集成任何第三方 SDK**, 包括但不限于:

- ❌ 统计 / 分析 SDK (如 Google Analytics、友盟、百度统计、Mixpanel)
- ❌ 崩溃 / 错误上报 SDK (如 Firebase Crashlytics、Sentry、Bugly)
- ❌ 广告 SDK (如 AdMob、穿山甲、优量汇)
- ❌ 社交分享 SDK (如友盟分享、ShareSDK)
- ❌ 推送 SDK (如极光推送、个推、Firebase Cloud Messaging)
- ❌ 支付 SDK

如果您将来看到本应用集成上述任何 SDK, 请立即通过本政策第十条所列方式与我们联系, 确认是否为官方版本。

---

## 六、申请的 Android 权限

本应用当前版本**不申请任何 Android 权限**。

未来版本中, 如需新增以下功能, 将申请对应权限 (届时本政策将更新, 并在新版本中向您说明):

| 计划功能 | 计划申请的权限 | 用途说明 | 用户控制 |
|---|---|---|---|
| 本地通知 (提醒开始) | `POST_NOTIFICATIONS` | 在合适时间提醒您开始一件新的事 | 您可在系统设置 → 应用 → 粒时 → 通知 中随时关闭 |

我们**不会**申请以下权限 (永久):

- ❌ 定位权限 (`ACCESS_FINE_LOCATION` / `ACCESS_COARSE_LOCATION`)
- ❌ 相机权限 (`CAMERA`)
- ❌ 麦克风权限 (`RECORD_AUDIO`)
- ❌ 通讯录权限 (`READ_CONTACTS`)
- ❌ 短信权限 (`READ_SMS` / `SEND_SMS`)
- ❌ 存储权限 (本应用数据存储于应用沙箱, 不需外部存储权限)
- ❌ 电话权限 (`READ_PHONE_STATE`)
- ❌ 后台运行权限 (除本地通知外)

## 七、用户权利

根据《中华人民共和国个人信息保护法》及相关法律法规, 您对自己的数据享有以下权利:

### 7.1 访问权

您可以在应用内的"提交记录"页面随时查看您创建的所有数据。

### 7.2 删除权

您可以通过以下方式删除您的数据:

- 在应用内删除单条记录 (未来版本将提供)
- 清除应用数据 (系统设置 → 应用 → 粒时 → 存储 → 清除数据)
- 卸载应用 (永久删除所有数据)

### 7.3 导出权 (未来)

未来版本可能提供"导出数据"功能, 让您将本地数据导出为通用格式 (如 JSON、CSV)。

### 7.4 撤回同意

您可以随时通过卸载本应用撤回对本政策的同意, 撤回后我们将不再处理您的信息 (因我们从未处理过)。

### 7.5 投诉举报

如您认为本应用违反法律法规或本政策约定收集使用您的个人信息, 您可以通过本政策第十条所列方式联系我们, 或向网信、市场监督管理部门投诉举报。

## 八、未成年人保护

### 8.1 适用人群

本应用面向**一般用户**设计, 不针对 14 岁以下儿童。

### 8.2 未成年人使用

如您是未成年人 (14 周岁以上, 18 周岁以下), 在使用本应用前, 请由您的监护人代为仔细阅读本协议和本政策, 并在征得监护人同意后使用本应用。

如您是 14 周岁以下的儿童, 请**不要**使用本应用。

由于本应用不收集个人信息, 上述保护措施对您的影响有限, 但我们仍作出此说明以示审慎。
