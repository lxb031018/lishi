## 一、关于本应用

**粒时** 是一款离线个人时间管理工具, 灵感来自 Git 的"原子提交"理念, 鼓励用户在自由时间主动做点有意义的事并记录下来。

### 1.1 核心功能

- 用户写下想做的事情, 点击开始后进入计时
- 计时完成后保存为一次"提交" (类似 git commit)
- 提交后进入 5 分钟休息倒计时
- 展示历史提交记录, 支持按日期分组查看

### 1.2 应用特点

- **无需注册账号**
- **不联网** (无任何后端服务)
- **不收集任何用户数据**
- **不包含广告**
- **不收取任何费用**
- 全部数据存储于用户设备本地 (Android 应用沙箱目录)
